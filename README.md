# AI_Challenge_Banking-PSBxMAI

## Функции
*  Анализ входящей корреспонденции
*  Определение критичных параметров обработки
*  Генерация адаптивных ответов 
*  Интеллектуальная маршрутизация и согласование
*  Интеллектуальная система уведомлений

## Алгоритм проекта
- Регистрация в системе
- Импортируется письмо
- AI модель анализирует содержание письма, классифицирует и выделяет ключевые темы, параметры обработки
- Генерируется адаптивный ответ с учётом параметров и ключевых тем
- Письмо, запрос и ответ заносятся в базу данных
- Приходят оповещения в системе/email
