# AI_Challenge_Banking-PSBxMAI

Этот проект представляет собой интеллекутальную систему генерации деловой переписки корпоративного уровня, которая обеспечит сквозную автоматизацию процесса создания, согласования и отправки официальных писем. Система должна стать единой плат- формой для работы с внешней корреспонденцией, гарантирующей высочайшее качество контента, безусловное соответствие корпоративным стандартам и пол- ное соблюдение регуляторных требований при значительном сокращении вре- менных затрат.

### 1\. Подготовка окружения (Python)

Перед началом убедитесь, что у вас установлен Python (версии 3.10 или выше).

1.  **Создайте и активируйте виртуальное окружение (`venv`):**
    ```bash
    python -m venv venv
    venv\Scripts\activate
    ```
2.  **Установите зависимости Python:**
    ```bash
    (venv) pip install -r requirements.txt
    ```
    *Если файл `requirements.txt` отсутствует, выполните:*
    ```bash
    (venv) pip install uvicorn fastapi sqlalchemy[asyncio] asyncpg python-dotenv gigachat python-multipart python-docx pypdf requests
    ```

### 2\. Подготовка окружения (JavaScript)

1.  Установите зависимости для фронтенда с помощью менеджера пакетов `npm`. Убедитесь, что вы находитесь в корневой папке проекта:
    ```bash
    npm install
    ```

-----

## 🔑 Конфигурация API (Файл `.env`)

Для работы с GigaChat API необходимо настроить файл переменных окружения.

1.  Создайте файл с именем **`.env`** в корневой папке проекта.
2.  Добавьте в него ваш ключ доступа к API GigaChat:
    ```env
    GIGACHAT_API_KEY="ВАШ_КЛЮЧ_ЗДЕСЬ"
    ```

-----

## Запуск проекта

Проект запускается в двух отдельных консолях.

### 1\. Запуск Бэкенда (API)

Запустите бэкенд-сервер Uvicorn. **Обязательно** убедитесь, что виртуальное окружение `(venv)` активно.

```bash
(venv) uvicorn backend.main:app
```

Сервер будет доступен по адресу: **`http://127.0.0.1:8000`**

### 2\. Запуск Фронтенда (Веб-интерфейс)

Откройте **новое окно консоли** и запустите фронтенд:

```bash
npm run frontend
```

Приложение будет доступно в браузере (обычно) по адресу: **`http://localhost:5173`**
